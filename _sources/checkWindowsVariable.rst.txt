Перевірка змінних оточення Windows
==================================
**Необхідно, щоб хочаб одна з двух була вказана:**

- Відкрийте командний рядок (WIN+R):

          .. image:: /Images/windowsVariable/1.jpg

- Введіть в командному рядку команди:

          .. image:: /Images/windowsVariable/2.jpg

- Перейдіть до налаштування системи:

          .. image:: /Images/windowsVariable/3.jpg

- Перейдіть до Додаткові налаштування системи:

          .. image:: /Images/windowsVariable/4.jpg

- На вкладці "Додатково" оберіть "Змінні оточення":

          .. image:: /Images/windowsVariable/5.jpg

**Додавання шляху до Java в змінну PATH:**
  
          .. image:: /Images/windowsVariable/6.jpg
          .. image:: /Images/windowsVariable/7.jpg

**Додавання шляху до Java в змінну JAVA_HOME:**
  
          .. image:: /Images/windowsVariable/8.jpg
          .. image:: /Images/windowsVariable/9.jpg        
          .. image:: /Images/windowsVariable/10.jpg

**Після внесення змін до PATH і JAVA_HOME необхідно перелогінитись**
